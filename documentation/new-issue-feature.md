# New Issue Submission Feature

This feature allows users to create and submit new issues directly within the application. Users can report bugs, suggest new features, or provide feedback seamlessly.

## How to Use

1. Navigate to the `/new-issue` page.
2. Fill out the issue form with a title and description.
3. Submit the form to create a new issue.

The submitted issues are stored in the database and can be managed by the development team.
