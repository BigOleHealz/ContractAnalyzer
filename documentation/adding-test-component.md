# Adding Test Component

This document describes how to add and view the `TestComponent` in the application.

## Steps to Run the Application

1. **Install Dependencies**: Run `npm install` to install all necessary packages.
2. **Start the Development Server**: Use `npm run dev` to start the development server.
3. **View the Test Component**: Navigate to `http://localhost:3000/test` to view the `TestComponent`.
4. **Run Tests**: Execute `npm test` to run the test suite and ensure everything is working correctly.
