# Scrum Features Documentation

## Overview
This document provides an overview of the Scrum-based project management tools integrated into the application.

## Features
- **Backlog**: Manage and prioritize tasks.
- **Sprint Planning**: Plan and organize sprints.
- **Daily Stand-ups**: Facilitate daily meetings to discuss progress.
- **Retrospectives**: Review and improve the process after each sprint.

## Usage
Navigate to the respective sections via the application menu to access and utilize these features.
